# Система онбординга - Frontend

## Описание

Современное веб-приложение для системы онбординга новых сотрудников, построенное на Vue.js 3 с использованием современных технологий и лучших практик разработки.

## Реализованный функционал

###  Аутентификация
- Страница входа (с возможностью интеграции SSO)
- Управление сессией пользователя
- Ролевая модель доступа

###  Дашборд
- Прогресс-бар онбординга
- Список модулей с статусами
- Визуализация прогресса по обязательным и рекомендуемым модулям

###  Модули онбординга
- Просмотр контента модулей
- Отметка модулей как прочитанных
- История попыток прохождения тестов

###  Тестирование
- Прохождение тестов по модулям
- Случайный выбор вопросов из банка
- Валидация ответов
- Ограничение количества попыток
- Детальные результаты с разбором ответов

###  Отчёты
- Отчёт по прогрессу онбординга сотрудника
- Отчёт по результатам тестов
- Отчёт по подразделению (для руководителей)
- Фильтрация и экспорт данных

###  Администрирование
- Управление модулями
- Управление пользователями
- Управление вопросами тестов
- CRUD операции для всех сущностей

###  Профиль пользователя
- Информация о сотруднике
- Статус онбординга
- Интеграция с ПСО (заглушка)

##  стек технологий 

- **Vue 3** (Composition API)
- **Vue Router 4** - маршрутизация
- **Pinia** - state management
- **Vite** - сборщик
- **Tailwind CSS** - стилизация
- **Axios** - HTTP клиент

## Структура проекта

```

├── src/
```

## Роли пользователей

1. **Новый сотрудник** - проходит модули и тесты
2. **Наставник** - видит прогресс подопечных
3. **HR-специалист** - управляет пользователями и формирует отчёты
4. **Руководитель подразделения** - видит отчёты по своему подразделению
5. **Администратор системы** - полный доступ к управлению

## Особенности реализации

### UI/UX
- Современный и интуитивный интерфейс
- Адаптивный дизайн
- Прогресс-бары и визуализация данных
- Цветовая индикация статусов
- Плавные переходы и анимации

### Безопасность
- Ролевая модель доступа
- Защита маршрутов
- Валидация данных на клиенте

### Производительность
- Ленивая загрузка компонентов
- Оптимизированные API запросы
- Кэширование данных в stores

## Интеграции

### Бэкенд API
- Полная интеграция с REST API бэкенда
- Обработка ошибок
- Автоматическая обработка ответов

### SSO (планируется)
- Заглушка для интеграции с корпоративным SSO
- Готовность к интеграции с LDAP/Active Directory

### ПСО (планируется)
- Заглушка для интеграции с системой запросов доступов
- Передача данных пользователя в ПСО

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для production
npm run build
```

## Конфигурация

### API Endpoint
Настраивается в `vite.config.js`:
```javascript
server: {
  proxy: {
    '/api': {
      target: 'http://localhost:5000',
      changeOrigin: true
    }
  }
}
```

### Цветовая схема
Настраивается в `tailwind.config.js`
