# Быстрый старт

## Установка и запуск


   ```bash
   cd onboarding-frontend
   npm install
   ```

   - Бэкенд должен быть доступен на `http://localhost:5000`
   - Или измените proxy в `vite.config.js`


   ```bash
   npm run dev
   ```


   - Приложение будет доступно на `http://localhost:3000`

## Первый вход

1. На странице входа введите ID пользователя из базы данных
2. После входа вы увидите дашборд с вашим прогрессом онбординга

## Основные функции

### Для нового сотрудника:
- Просмотр модулей онбординга
- Прохождение тестов
- Отслеживание прогресса

### Для наставника:
- Просмотр прогресса подопечных
- Отчёты по результатам тестов

### Для HR:
- Управление пользователями
- Формирование отчётов
- Администрирование модулей

### Для администратора:
- Полный доступ к управлению системой
- Создание и редактирование модулей
- Управление вопросами

## Структура проекта

- `src/views/` - Страницы приложения
- `src/components/` - Переиспользуемые компоненты
- `src/stores/` - State management (Pinia)
- `src/api/` - API сервисы
- `src/router/` - Маршрутизация

## Настройка

### Изменение API endpoint

Отредактируйте `vite.config.js`:

```javascript
server: {
  proxy: {
    '/api': {
      target: 'http://your-backend-url:port',
      changeOrigin: true
    }
  }
}
```

### Изменение цветовой схемы

Отредактируйте `tailwind.config.js` для изменения цветов интерфейса.

## Troubleshooting

### Ошибка подключения к API
- Убедитесь, что бэкенд запущен
- Проверьте настройки proxy в `vite.config.js`
- Проверьте CORS настройки на бэкенде

### Ошибки при сборке
- Удалите `node_modules` и `package-lock.json`
- Выполните `npm install` заново

### Проблемы со стилями
- Убедитесь, что Tailwind CSS правильно настроен
- Проверьте `postcss.config.js`
